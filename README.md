# eloth-html
Eloth mais en HTML.

## Documentation
### Qu'est ce que c'est ?
C'est un site qui aide pour le jeu de rôle sur table : Eloth !
### Essayer le site localement
1. Télécharger Docker
2. Envoyer cette commande dans le Terminal:
`docker build -t eloth-html . && docker run -d -p 8080:80 eloth-html`

3. Accéder à `http://localhost:8080` dans un navigateur.
